# 새싹

문제링크 : https://www.acmicpc.net/problem/25083

## 문제

아래 예제와 같이 새싹을 출력하시오.

## 입력

입력은 없다.

## 출력 

새싹을 출력한다.

```python
print("         ,r'\"7")
print("r`-_   ,'  ,/")
print(" \. \". L_r'")
print("   `~\/")
print("      |")
print("      |")
```

문자열로 묶기 위해 '""' 를 사용할 때에, 문자열 안에 있는 '"'값을 구현하기 위해 \ 를 사용했습니다.
