# [기초-비트시프트연산] 2의 거듭제곱 배로 곱해 출력하기

[문제링크](https://codeup.kr/problem.php?id=6047)



## 1. 문제설명

정수 2개(a, b)를 입력받아 a를 2b배 곱한 값으로 출력해보자.
0 <= a <= 10, 0 <= b <= 10




## 2. 문제풀이

```python
a, b =input().split()
a = int(a)
b = int(b)
print(a<<b)
```
