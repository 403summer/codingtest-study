# [BOJ] 1085_직사각형에서 탈출

문제 링크 : https://www.acmicpc.net/problem/1085

-------------
## 문제 요약
  - 주어진 좌표 (x, y)에서 직사각형의 모서리까지 가는 거리의 최솟값을 구하는 문제

## 접근 방식
  - 좌표에서 모서리에 수직으로 이동하는 것이 가장 짧다.
  - 네 방향 중에서 가장 짧은 거리를 찾으면 된다.
