# 고양이

문제링크 : https://www.acmicpc.net/problem/10171

## 문제

아래 예제와 같이 고양이를 출력하시오.

## 입력

없음.

## 출력 

고양이를 출력한다.

```python
print("\    /\ ")
print(" )  ( ') ")
print("(  /  ) ")
print(" \(__)|")
```

출력문은 1개당 한 줄씩 출력되는걸 이용해서 print를 여러 개 써서 출력했습니다. 

```python
print("\    /\\\n )  ( ')\n(  /  )\n \(__)|")
```

\ 를 두 번 써서 줄바꿈을 먹이는 걸 통해 한 줄로도 작성 가능합니다.
